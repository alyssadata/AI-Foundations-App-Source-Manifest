This repository defines the universal app source manifest for any app, interface, blank AI, companion surface, bounded workspace, or product layer that references AI Foundations / Origin | Continuum.

The universal app source manifest exists to ensure that AI Foundations is referenced only through source-line preservation, bounded workspace containment, and non-extension.
